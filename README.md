# n8n-automated-linkedin-bot# 🤖 n8n LinkedIn Post Automation
An automated pipeline that researches tech topics, generates AI images, and schedules LinkedIn posts.

### 🛠 Tech Stack
* **Automation:** [n8n](https://n8n.io/)
* **Research:** Tavily AI API
* **LLM:** Google Gemini (via LangChain nodes)
* **Image Gen:** Custom Worker API / Google Creative Director prompts
* **Destination:** LinkedIn API
